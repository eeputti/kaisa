# kaisa (static)

A single-page, static HTML site (no build, no frameworks).  
Works on **GitHub Pages** and **Vercel** as-is.

## Edit
- Text/content: `index.html`
- Style: `styles.css`
- Main image or gif: replace `media/main.jpg` (you can use a GIF but keep the filename)

## GitHub Pages
1. Push these files to your repo root.
2. GitHub → Settings → Pages
3. Source: `Deploy from a branch`
4. Branch: `main` and folder: `/ (root)`
5. Save → wait → open the Pages URL.

## Vercel
1. Import the repo in Vercel.
2. Framework preset: **Other**
3. Build command: (leave empty)
4. Output directory: (leave empty)
5. Deploy.

That’s it.
