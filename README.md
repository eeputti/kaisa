# kaisa-onepage

Minimal + vintage one-page site (Next.js) inspired by clean editorial layouts.

## Edit content
- `content/site.ts` = all text + links + image paths
- `public/media/main.jpg` = swap to your own photo (or change to `main.gif` and update content)
- `app/globals.css` = style tweaks

## Run locally
```bash
npm install
npm run dev
```

Open http://localhost:3000

## Deploy to Vercel
- Push to GitHub
- Vercel → New Project → import repo → Deploy
